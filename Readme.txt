To-Do List Project
A feature-rich To-Do List application designed with a user-friendly interface, allowing users to efficiently manage their tasks by setting priorities, due dates, and marking completed tasks. This project demonstrates my skills in front-end development using HTML, CSS, and JavaScript.

Features-
User-friendly Interface: Clean and modern design for easy navigation.
Task Management: Add, edit, and delete tasks with ease.
Priority Levels: Assign priority levels to tasks (High, Medium, Low) for better organization.
Due Dates: Set due dates for tasks to ensure timely completion.
Completion Status: Mark tasks as complete and easily filter completed tasks.
Responsive Design: Fully responsive layout for seamless use on all devices.
Technologies Used
HTML
CSS
JavaScript
